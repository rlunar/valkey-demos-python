# Valkey Python Demos

A collection of Python demonstrations showcasing Valkey features and use cases.

## Demos

### Credit Card Fraud Detection
Real-time fraud detection system using Valkey for data processing and caching.

**Location:** `creditcardfraud/`

**Dependencies:**
- Python 3.12+
- Valkey
- NumPy
- Pandas

### Valkey 9 Features
Demonstrations of new features introduced in Valkey 9.

**Location:** `valkey-9-features/`

**Dependencies:**
- Python 3.12+

## Getting Started

Each demo is self-contained in its own directory with its own dependencies and README.

Navigate to the specific demo directory and follow the instructions there.

## Requirements

- Python 3.12 or higher
- [uv](https://docs.astral.sh/uv/) package manager (recommended)
